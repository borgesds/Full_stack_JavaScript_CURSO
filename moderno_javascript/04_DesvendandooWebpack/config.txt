--INSTALAR O WEBPACK

npm install --save-dev webpack webpack-cli

--Install a biblioteca moment

npm install moment

--Install 

npx webpack

--URGENT: 
O npx acima é só para execução do exercício. Depois de criar um arquivo webpack.config.js, deve excluir o main.js se vc executou o codigo acima se não so criar dentro do novo arquivo:

module.exports = {
    entry: {
        index: './src/index.js'
    }
}

Execute:
npx webpack --mode development

const path = require('path')

    }
}

npx webpack --mode development
