NPX e NPM Scripts

Um atalho para poder converter os arquivos JS para versões anteriores: npx babel nomeDoArquivo.js -o nomeDoArquivoNovo.js

Criar um arquivo padrão para converter os arquivos de forma automática, o codigo fica dentro do package;json: "build": "npx babel index.js -o transpiled.js" Para rodar ele digite no terminal: npm run build
